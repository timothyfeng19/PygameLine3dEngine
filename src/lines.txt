Each line should follow the format x1 y1 z1 x2 y2 z2 which is starting point, ending point.
X is left to right, Y is down to up, Z is backward to forward.
0 0 0 is the starting player point.
Only supports integers currently (too lazy to change for now).
Each line should have a space after the last integer, and start below.
-1 -1 1 1 -1 1 
0 -1 1 0 -1 0
